# 対応法令プリセット

内部 registry に同梱されている 41 法令の `law_id` と略称の一覧です。`resolve_law` はこの registry を最優先で参照します。

registry にない法令でも、`resolve_law` は e-Gov 検索の厳密一致から候補補完することがあります。ただし曖昧入力を勝手に 1 件採用することはしません。

## 委任関係

`find_related_sources` および `get_evidence_bundle` は、以下の明示的な委任関係も参照します。

| 親法 | 委任先 |
|---|---|
| 労働基準法 | 労働基準法施行令、労働基準法施行規則 |
| 労働安全衛生法 | 労働安全衛生法施行令、労働安全衛生規則 |
| 雇用保険法 | 雇用保険法施行令、雇用保険法施行規則 |
| 労働者災害補償保険法 | 労働保険の保険料の徴収等に関する法律 |
| 労働保険の保険料の徴収等に関する法律 | 労働者災害補償保険法、雇用保険法 |

加えて、命名規則（`〇〇法` → `〇〇法施行令` / `〇〇法施行規則`）からの自動推論も行います。

## registry にない法令

`resolve_law` は e-Gov 検索 API へのフォールバックを行いますが、曖昧一致での自動採用は抑止されます。registry 追加が望ましい法令にお気づきの場合は GitHub issue にてご連絡ください。
